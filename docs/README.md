# RTC - Real Time Chat Application

## Installation

### Prérequis
- Rust + Cargo
- Node.js + npm
- PostgreSQL
- MongoDB

### Backend
\\\ash
cd server
cargo build
cargo run
\\\

### Frontend
\\\ash
cd client
npm install
npm run dev
\\\
